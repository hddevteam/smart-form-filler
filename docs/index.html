Quick Installation

Get started in less than 30 seconds

1

Download Extension

Get the latest release from GitHub and extract the extension package to your local machine.

2

Enable Developer Mode

Open Chrome/Edge extensions page and enable "Developer mode" in the top right corner.

3

Load Extension

Click "Load unpacked" and select the extracted extension folder to install it in your browser.

4

Configure & Start

Set up your preferred AI model (Ollama or cloud) and start filling forms automatically!
