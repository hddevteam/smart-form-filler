üì¢ Important for Local File Access

If you opened this demo via file:// URL, the extension needs special permissions:

  1. Go to chrome://extensions/ (or edge://extensions/)
  2. Find "Smart Form Filler" extension
  3. Click "Details"
  4. Enable "Allow access to file URLs"

Recommended: Use npm run demo to run the demo server at http://localhost:3002
